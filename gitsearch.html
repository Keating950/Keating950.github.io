Gitsearch
 

Gitsearch is a simple command line program for searching for and cloning repos from Github. I wrote the program in Python, developing the interface in curses and using GitHub's public API for the search and clone features. Given the difficulties of Python packaging and dependency management, I strived to minimize external dependencies; Gitsearch has only one, the highly popular Requests library.
